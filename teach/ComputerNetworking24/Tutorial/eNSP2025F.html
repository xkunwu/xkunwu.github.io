Course Information

Short Intro

This course provides practical tutorials for Computer Networking, based on eNSP software from Huawei.

Prerequisites

We expect you to have the following skills before taking this course:

Required：

  • Computer Networking

Recommended：

  • Comfortable with interactive shells

Teaching Plan

We organize this course into a 8-week session (4 hours per week). The main contents are listed below:

  1. 办公室网络搭建
    1. 认识网络通信
    2. 终端网络设置
    3. 终端网络安全
  2. 写字楼网络搭建
    1. VLAN通信配置
    2. 解决交换网络环路问题
    3. 链路聚合配置
    4. 网络设备管理
    5. DHCP初级应用
    6. 网络设备故障排除
  3. 智慧校园网络搭建
    1. IP地址划分
    2. IP地址分配
    3. WLAN的规划设计
    4. 无线网络设置
    5. WLAN安全和维护
  4. 大型企业网络搭建
    1. 静态路由设置
    2. 动态路由设置
    3. 外网通信配置
    4. 跨地域通信配置
  5. 网络安全与网络管理
    1. 数据传输安全
    2. ACL配置
    3. 防火墙配置
    4. 网络系统管理

Methodology

Problem-solving-oriented, with a primary emphasis on practical skills.

以解决实际问题为导向，主要强调实践技能。
